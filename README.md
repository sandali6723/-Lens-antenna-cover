# Lens Antenna Cover

## Overview
This project focuses on designing a **lens antenna cover** for a **conical horn antenna** using FreeCAD.  
The purpose of this design is to protect the antenna from dust, insects, and environmental particles while maintaining proper microwave antenna performance.

The antenna cover was modeled and dimensioned based on microwave antenna size calculations and mechanical fitting requirements.

---

## Project Objectives
- Design a protective cover for a conical horn antenna
- Prevent dust and insects from entering the antenna
- Create a lightweight and practical microwave antenna enclosure
- Develop the model using FreeCAD for easy modification and future improvements

---

## Software Used
- FreeCAD
- CAD Modeling and Dimensioning Tools
